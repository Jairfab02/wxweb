LEIA-ME PARA USUARIOS FIVEWIN
================================================================================
Pré-requisitos para utilizar a wxWeb.

* Harbour
  É necessário ter compilador Harbour mais recente instalado em sua CPU com su-
  porte à HBMK2 para compilação e linkagem dos codigos fontes. É possível baixar
  uma cópia mais atual diretamente dos servidores da FiveTech na seguinte URL:
  
  http://www.fivetechsoft.com/files/harbour.exe

* Borland C++ Compiler
  É necessário ter o compilador borland C++ instalado, configurado e pronto para
  uso antes de se compilar os exemplos da wxWeb. O procedimento de como obter,
  instalar e/ou configurar o BCC está além do escopo deste documento e pois já
  existem várias informações sobre este tópico na web.

* Apache
  Nesta versão a wxWeb gera aplicativos CGI. Isto requer que você tenha acesso
  à um servidor web previamente já instalado e configurado com suporte à CGI
  para que você possa testar os aplicativos gerados. Existem vários servidores
  web comerciais ou open-source tais como IIS, Apache, Xitami e muitos outros.
  
  Embora o procedimento de como obter, instalar e/ou configurar um servidor web
  esteja além do escopo deste documento, gentilmente incluimos nesta na pasta um
  pequeno tutorial sobre como baixar, instalar e configurar o Apache. Consulte o
  arquivo HowToConfigureApache.txt... meus agradecimentos ao Rossine pela ajuda
  em montar este tutorial.
