TimeDiffAsSeconds()
Retorna a diferença em segundos entre duas datas e/ou dois horários fornecidos.
 

Sintaxe:
TimeDiffAsSeconds( <dDateStart>, <dDateEnd>, [<cTimeStart>], [<cTimeEnd>] ) -> nSeconds
 

Argumentos:
<dDateStart>
Data inicial para comparação.
 
<dDateEnd>
Data final para comparação
 
<cTimeStart>
Horário inicial para comparação. Será assumido '00:00:00' se omitido.
 
<cTimeEnd>
Horário final para comparação. Será assumido '00:00:00' se omitido.
 

Descrição:
Retorna a diferença em segundos entre duas datas e/ou dois horários fornecidos.
