wxGetClientUserAgent()
Retorna uma string informando o tipo de navegador instalado no cliente com base no conteúdo do header User-Agent da requisição atual, se houver.
 

Sintaxe:
wxGetClientUserAgent( )
 

Descrição:
Retorna uma string informando o tipo de navegador instalado no cliente com base no conteúdo do header User-Agent da requisição atual, se houver. Esta função tenta idenfificiar o navegador do usuário ou melhor, o "agente de usuário" pelo qual a página está sendo acessada.

Um exemplo típico do retorno desta função seria algo como: 
 Mozilla/4.5 [en] (X11; U; Linux 2.2.9 i586) 
 Mozilla/4.0 (compatible; MSie 5.01; Windows 98) 
 Opera/7.54 (Windows NT 5.1; U) [en]

Além de outras coisas, você pode utilizar o retorno desta função para personalizar a geração de suas páginas para as capacidades do agente do usuário que efetuou a requisição.
