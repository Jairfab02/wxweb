wxOperaVersion()
Retorna uma string com o número da versão do navegador Opera em uso pelo cliente que efetuou a requisição atual.
 

Sintaxe:
wxOperaVersion( )
 

Descrição:
Retorna uma string com o número da versão do navegador Opera em uso pelo cliente que efetuou a requisição atual. Caso o cliente não esteja utilizando o Opera será retornado uma string vazia.
