SESSION()
Esta função cria, modifica ou exclui o conteudo de uma variável de sessão para a conexão atual.
 

Sintaxe:
Session( [ <pConnection>, ] <cKeyName> [, <xValue> ] ) -> xOldValue
 

Argumentos:
<pConnection>
É um parametro (pointer) que identifica a conexão à qual desejamos operar. Este parametro deverá ser omitido em circunstâncias normais, pois seu uso torna-se mais evidente em ambientes multi-thread ou que utilizem servidores de dados personalizados.
 
<cKeyName>
É o nome da variavel de sessão que desejamos consultar/alterar/excluir.
 
<xValue>
É um parâmetro opcional que contém o valor que deverá ser preservado para esta variável de sessão. Atribuindo o valor NIL para uma variavel de sessão existente forçará a mesma a ser removida.

A wxWeb suporta que sejam guardados vários tipos de valores dentro de uma variável de sessão tais como texto, números, data, valores lógicos, memos, objetos e arrays.
