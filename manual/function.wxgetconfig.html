WXGETCONFIG()
Puxa um valor carregado do arquivo de configuração para esta conexão.
 

Sintaxe:
wxGetConfig( [<pConn> ,] <cFieldName>, [<cDefaultValue>], [ <cNewValue>] ) -> cOldValue/cNewValue
 

Descrição:
Puxa um valor carregado do arquivo de configuração para esta conexão. Se <pConn> for omitido a conexão ativa baseada na thread atual será utilizada.
