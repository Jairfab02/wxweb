wxGetDeviceRes()
Tenta detectar o tamanho da tela do dispositivo móvel utilizado pelo cliente para acessar este script.
 

Sintaxe:
wxGetDeviceRes( )
 

Descrição:
Tenta detectar o tamanho da tela do dispositivo móvel utilizado pelo cliente para acessar este script. Se a informacao estiver disponível retorna um array informando o tamanho da LARGURA e ALTURA da tela em Pixels, caso contrário retorna um array vazio.
