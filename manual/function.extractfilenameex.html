ExtractFileNameEx()
Retorna apenas o nome do arquivo contido em cFileName juntamente com seu PATH (se estiver disponível) mas sem extensão.
 

Sintaxe:
ExtractFileNameEx( <cFileName> ) -> cName
 

Argumentos:
<cFileName>
A string contendo o nome do arquivo que será processado.
 

Descrição:
Retorna apenas o nome do arquivo contido em cFileName juntamente com seu PATH (se estiver disponível) mas sem extensão.

Exemplo :

 
 #include "wxweb.ch"  
 
 FUNCTION main()  
 
       cDbf := 'c:\data\estoque.dbf'  
       cCdx := ExtractFileNameEx( cDbf ) + '.cdx'  
 
       USE ( cDBF ) NEW ALIAS est  
 
       IF !File( cCdx )  
          INDEX ON ...  
       End  
 
    RETURN nil
