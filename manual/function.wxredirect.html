WXREDIRECT()
Envia um cabeçalho HTTP forçando o redirecionamento do navegador para outra página.
 

Sintaxe:
wxRedirect( [<pConn> ,] <cUrl> [, <cParams | aParams>] ) -> nil
 

Argumentos:
<pConn>
O pointer da conexão a ser utilizada para envio. Se omitido a conexão ativa baseada na thread atual será utilizada.
 
<cUrl>
A URL de destino para onde desejamos redirecionar o usuário.
 
<cParams|aParams>
Este argumento poderá ser uma string contendo um único parâmetro à ser enviado juntamente com URL de destino via método GET. A wxWeb suporta adicionalmente que seja passado um array (podendo ser bidimensional ou não) contendo inúmeros parâmetros à serem enviados.
 

Descrição:
Envia um cabeçalho HTTP forçando o redirecionamento do navegador para outra página.
 

Info:
Source: W:\rtl\Kernel\wxConnManager.c
Revisado em: 22/11/2006 as 15:27:16
 

Exemplo :

 
       wxRedirect( 'www.google.com.br' )  
 
       wxRedirect( 'pagina2', 'nome=Renato' )  
 
       wxRedirect( 'pagina2', {'nome=Renato','Idade=27','Married=.t.'} )  
 
       wxRedirect( 'pagina2', {{'nome','Renato'},{'Idade','27'},{'Married', '.t.'}} )
