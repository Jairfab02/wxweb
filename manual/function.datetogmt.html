DateToGMT()
Formata a uma data e hora para uma string no formato GMT.
 

Sintaxe:
DateToGMT( [<dDate>], [<cTime>], [<nDaysToAdd>], [<nSecsToAdd>] ) -> cGMTDate
 

Argumentos:
<dDate>
A data a ser convertida. Será assumido a data atual obtida com a função Date() caso seja omitido.
 
<cTime>
A hora no formato HH:MM[:SS]. Se omitido será assumido a hora e atual obtida com Time().
 
<nDaysToAdd>
Adiciona em <dDate> a quantidade de dias informados à data antes da conversão. Pode ser passado um valor negativo neste parâmetro.
 
<nSecsToAdd>
Informa a quantidade de segundos à serem acrescidos à <dDate> e <cTime> antes da conversão.
 

Descrição:
Formata a uma data e hora para uma string no formato GMT.
 

Info:
Source: W:\rtl\Kernel\Sources\wxCgifunc.prg
 

Exemplo :

 
 #include "wxweb.ch"  
 
 FUNCTION main()  
 
    dHoje := date()               // 10/31/2003  
 
    ? DateToGMT( dHoje )          // exibe "Saturday, 31-Oct-03 00:00:00 GMT".  
 
    RETURN nil
