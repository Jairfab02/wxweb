WXCONNECTION_CREATE()
Cria uma nova conexão e inicializa os seus valores default.
 

Sintaxe:
wxConnection_Create( <nHandle>, <cOutPutFunc> ) -> pConn
 

Argumentos:
<nHandle>
Um valor numerico (Handle) para identificar a conexão atual dentro da pilha de conexões gerenciadas pela wxWeb.
 
<cOutPutFunc>
Uma string contendo o nome da função que irá operar sobre toda saida de tela que ocorrer durante o período que esta conexão estiver ativa.
 

Descrição:
Cria uma nova conexão e inicializa os seus valores default.
