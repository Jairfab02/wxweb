WXGETDEFAULTPARAM()
Verifica se existe algum argumento passado em linha de comando, que possa ser identificado como sendo um 'parâmetro default' em uma requisição Web.
 

Sintaxe:
wxGetDefaultParam( [ <pConn> ] ) -> cDefaultValue
 

Descrição:
Verifica se existe algum argumento passado em linha de comando, que possa ser identificado como sendo um 'parâmetro default' em uma requisição Web. Se <pConn> for omitido a conexão ativa baseada na thread atual será utilizada.
 

Info:
Source: W:\rtl\Kernel\wxConnManager.c
Revisado em: 24/10/2008 as 19:46:48
 

Exemplo :

 
    http://localhost/Site.exe?downloads                --> downloads  
    http://localhost/Main.exe?pedidos&chave=999        --> pedidos  
    http://localhost/Site.exe?download&fileid=0151     --> download
