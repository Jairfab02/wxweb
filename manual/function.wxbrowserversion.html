wxBrowserVersion()
Retorna a versao atual do navegador em uso pelo cliente que efetuou a requisição atual.
 

Sintaxe:
wxBrowserVersion( )
 

Descrição:
Retorna a versao atual do navegador em uso pelo cliente que efetuou a requisição atual. Caso não seja possível determinar a versão do user agent em uso esta função retorna uma string contendo o termo "(Unknown)".
