wxIsChrome()
Retorna .T. se o usuario estiver usando o Google Chrome para efetuar a requisição atual.
 

Sintaxe:
wxIsChrome( )
 

Descrição:
Retorna .T. se o usuario estiver usando o Google Chrome para efetuar a requisição atual. Um exemplo do retorno de wxGetClientUserAgent() que demonstra o Google Chrome rodando em Windows seria: 
 Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US) AppleWebKit/525.13 (KHTML, like Gecko) Chrome/0.2.149.30 Safari/525.13
