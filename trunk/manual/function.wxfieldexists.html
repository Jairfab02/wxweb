WXFIELDEXISTS()
Retorna .T. se pelo menos um campo com o nome igual à <cFieldName> tiver sido recebido via método GET ou POST na conexão passada como argumento.
 

Sintaxe:
wxFieldExists( [ <pConn>, ] <cFieldName> ) -> lFound
 

Argumentos:
<pConn>
O pointer da conexão à ser utilizada para consulta. Se omitido a conexão ativa baseada na thread atual será utilizada.
 
<cFieldName>
Nome do campo HTML cujo nome iremos testar a existência.
 

Descrição:
Retorna .T. se pelo menos um campo com o nome igual à <cFieldName> tiver sido recebido via método GET ou POST na conexão passada como argumento.
