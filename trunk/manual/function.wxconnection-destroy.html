WXCONNECTION_DESTROY()
Destrói uma conexão previamente criada com wxConnection_Create().
 

Sintaxe:
wxConnection_Destroy( <pConn> ) -> nil
 

Argumentos:
<pConn>
O pointer para a conexão a ser destruida.
 

Descrição:
Destrói uma conexão previamente criada com wxConnection_Create(). Esta função envia quaisquer dados pendentes para a função de saida e libera os recursos de sistema atrelados à esta conexão.
