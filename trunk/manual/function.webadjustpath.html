WebAdjustPath()
Ajusta o PATH passado como parametro e asegura que a barra normal "/" seja o ultimo caracter da string passada.
 

Sintaxe:
WebAdjustPath( <cPath>, [<cDefault>] ) -> cPath
 

Argumentos:
<cPath>
A string contendo o PATH à ser analisado.
 
<cDefault>
Valor a ser retornado caso seja especificado e nenhuma informação esteja disponivel em <cPath>.
 

Descrição:
Ajusta o PATH passado como parametro e asegura que a barra normal "/" seja o ultimo caracter da string passada. Esta funcao nao altera nenhum dos caracteres '\' ou '/' existentes na string passada como argumento.
