wxIsIE()
Retorna .T. se o usuário que solicitou a requesição atual estiver usando o Microsoft Internet Explorer ou similar.
 

Sintaxe:
wxIsIE( )
 

Descrição:
Retorna .T. se o usuário que solicitou a requesição atual estiver usando o Microsoft Internet Explorer ou similar. Esta função faz uso do cabeçalho USER AGENT para tentar detectar o navegador em uso.

Possiveis valores para wxGetClientUserAgent() neste caso para Internet Explorer (Windows): 
  ie6 (on WinXP): 
  Mozilla/4.0 (compatible; MSie 6.0; Windows NT 5.1) 

  ie5.5 (on W2K): 
  Mozilla/4.0 (compatible; MSie 5.5; Windows NT 5.0) 

  ie5.01 (on Win98): 
  Mozilla/4.0 (compatible; MSie 5.01; Windows 98)
