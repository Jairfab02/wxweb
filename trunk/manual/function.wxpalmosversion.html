wxPalmOSVersion()
Retorna uma string com o número da versão do PalmOS instalada no cliente que efetuou a requisição atual.
 

Sintaxe:
wxPalmOSVersion( )
 

Descrição:
Retorna uma string com o número da versão do PalmOS instalada no cliente que efetuou a requisição atual. Caso o cliente não esteja utilizando o PalmOS será retornado uma string vazia.
