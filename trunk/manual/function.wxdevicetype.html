wxDeviceType()
Retorna o tipo de dispositivo móvel utilizado pelo cliente para navegação.
 

Sintaxe:
wxDeviceType( )
 

Descrição:
Retorna o tipo de dispositivo móvel utilizado pelo cliente para navegação. Caso não seja possivel detectar esta informação uma string vazia será retornada.
