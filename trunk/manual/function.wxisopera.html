wxIsOpera()
Retorna .T. se o usuario estiver usando o navegador Opera para efetuar a requisição atual.
 

Sintaxe:
wxIsOpera( )
 

Descrição:
Retorna .T. se o usuario estiver usando o navegador Opera para efetuar a requisição atual. Alguns possiveis valores para wxGetClientUserAgent() neste caso seriam (tanto em windows e linux): 
  Opera 7.54 (on WinXP): 
  Mozilla/4.0 (compatible; MSie 6.0; Windows NT 5.1) Opera 7.54 [en] 
  Opera/7.54 (Windows NT 5.1; U) [en] 

  (first example is spoofed as ie6, second without spoofing)
