wxClientIP()
Retorna o numero IP da estação atualmente conectada neste servidor ou uma string vazia caso nenhum IP tenha sido detectado.
 

Sintaxe:
wxClientIP( <lLoopBack> ) -> cIP
 

Argumentos:
<lLoopBack>
Caso não seja possivel determinar o IP atual da conexão atual, e este argumento seja setado como .F. esta função retornará uma string vazia. Se <lLoopBack> for omitido será assumido o valor .T. que fará com que o endereço IP padrão da interface LoopBack sejá retornado.
 

Descrição:
Retorna o numero IP da estação atualmente conectada neste servidor ou uma string vazia caso nenhum IP tenha sido detectado.

Exemplo 1:

 
 #include "wxweb.ch"  
 
 FUNCTION Main()  
 
    ? "Your IP address is:", wxClientIP()  
 
    RETURN nil

 #include "wxweb.ch"
