WXSENDHEADERDIRECT()
Envia um HEADER de baixo nivel ao navegador do usuario usando diretamente o ponteiro de conexão passado como primeiro argumento.
 

Sintaxe:
wxSendHeaderDirect( <pConn>, <cHeader> ) -> NIL
 

Argumentos:
<pConn>
O pointer para a conexão de destino para envio dos dados.
 
<cHeader>
O cabeçalho HTTP à ser enviado para a conexão.
 

Descrição:
Envia um HEADER de baixo nivel ao navegador do usuario usando diretamente o ponteiro de conexão passado como primeiro argumento. Devido à esta característica esta função tende a ser mais rápida em ambiente multi-threads pois evita qualquer tipo de busca tentando localizar a conexão ativa na pilha de conexões ativa.
