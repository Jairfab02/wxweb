ExtractFileExt()
Retorna apenas a extensão à partir de uma string contendo o nome do arquivo.
 

Sintaxe:
ExtractFileExt( <cFileName> ) -> cExt
 

Descrição:
Retorna apenas a extensão à partir de uma string contendo o nome do arquivo. Quaisquer informações sobre PATH ou NOME serão descartadas e inclusive o caracter de ponto que separa o nome do arquivo de sua extensão não será incluido no retorno desta função.
