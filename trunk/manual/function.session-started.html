SESSION_STARTED()
Retorna .T. se já fora iniciada uma sessão ou .F. caso nenhuma sessão tenha sido carregada ou iniciada.
 

Sintaxe:
Session_Started( [<pConn>] ) -> lStarted
 

Descrição:
Retorna .T. se já fora iniciada uma sessão ou .F. caso nenhuma sessão tenha sido carregada ou iniciada. Uma sessão é considerada como iniciada se ela já foi carregada de um arquivo para a memória ou se algum item foi atrelado à ela para posterior recuperação.
